VVVVVVVV NEW COMMANDS VVVVVVVV
touch {filename} | creates an empty file (built-in)  
echo {text} OR echo {text} > file | prints to terminal or writes to file (built-in)  
exit / quit | exits the shell (built-in)  
uptime | shows how long the shell has been running (built-in)  
history | prints previously used commands in order (built-in)  
alias name='command' | sets a quick alias you can use (built-in)  
say {text} | uses system speech to speak your input ( apt install pyttsx3 )  
calendar | shows current month calendar (built-in)
